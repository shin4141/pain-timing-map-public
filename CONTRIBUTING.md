# Contributing

Thank you for taking the time to improve Pain Timing Map.

This repo is currently a Manual MVP / Star-Presser v0. It is a public documentation surface for a manual workflow, not an implemented app or automation system.

## Accepted Contributions

Contributions are welcome when they improve:

- documentation clarity
- Signal Recheck examples
- typo fixes
- boundary clarification
- issue reports about ambiguous gates or confusing wording

## Not Accepted Without Decision Owner Approval

Do not open contribution work that adds or changes:

- implementation
- chart renderer
- API / scraping / automation
- pricing work
- new decision-material records
- public positioning changes
- release/tag work

## Boundaries to Preserve

Contributions should preserve:

- Noise-to-Evidence framing
- WATCH / HOLD / CAP / PREPARE / BUILD CANDIDATE gates
- BUILD CANDIDATE does not authorize implementation
- no prediction guarantee

Factual external claims should be cited or framed as pasted/hypothetical signals.
