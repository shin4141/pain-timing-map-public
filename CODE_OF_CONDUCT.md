# Code of Conduct

Pain Timing Map is for careful signal discussion and manual evidence framing.

Please keep discussion respectful and focused on the project boundary.

Do not:

- harass, abuse, doxx, or personally attack others
- post private personal data or sensitive scraped data
- turn the project into investment advice, hype pumping, or guaranteed prediction claims
- pressure maintainers or contributors into implementation, automation, pricing, release, or posting work outside the stated gates

The maintainer may close or remove issues and comments that violate these boundaries, create unsafe burden, or move the project away from its Manual MVP / Star-Presser v0 scope.
