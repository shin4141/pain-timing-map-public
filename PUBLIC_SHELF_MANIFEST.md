# Public Shelf Manifest

This repository is the public shelf for Pain Timing Map.

It contains public-safe documentation, examples, templates, schema notes, and usage boundaries for manually classifying pain signals before action.

## Public Shelf Contents

- `README.md`: public overview and start path.
- `docs/`: public-safe guides for scope, usage, visual flow, color tags, pain signal language, and downstream Entry intake packaging.
- `inputs/`: blank templates for manual review.
- `examples/`: public-safe examples and demos.
- `schema/`: optional structured record schema.
- `prompts/`: manual review prompt.
- Community files: contribution, conduct, support, and security boundaries.

## Excluded Materials

This public shelf excludes:

- non-public source artifacts,
- snapshots and restart-state materials,
- internal batches,
- registry content,
- source-only reports,
- private capture assets,
- pricing or sell-test materials,
- release materials,
